运算符

发表于 2017-06-20

算术运算符

加+、- 减、* 乘、/ 除、% 取模（求余数）、++、–
加号除了有算术运算的功能还有字符串拼接的功能

只有当两边都是数字的时候才会进行算术运算

只要又一边是字符串，那么就会把不是字符串的那边转换成字符串，然后进行拼接

++和–
前置和后置的区别

alert( 1+1 );//2        alert( 6/3 );//2    alert( 12%7 );//5

var i= 0；
i++;
相当于： i= i+ 1；
alert(i)//0
alert(i++);//0
++后置先使用，在增加

var i = 0;
var c = i++;        alert( c );//0        alert( i );//1

var i = 0;
var c = ++i;        alert( c );//0        alert( i );//1

赋值运算符

赋值运算符：

=、+=、-=、*=、/=、%=
a+=b 相当于 a = a + b
a = a + b可以简写成 a += b
注意 a = b + a不可以简写成 a += b

-=、*=、/=、%=的特性和+=一样

var i = 4;
        i += 3;
        相当于：
        i = i + 3;
        alert( i );

关系运算符

<、>、<=、>=、==、!=、===、!==
注意：返回值都是一个布尔值

alert( 1<3 )//true
<
如果左边小于右边则返回true,否则返回false

alert( 1<1 )//false
alert( 1<=1 )//true
<=
如果左边小于或等于右边则返回true,否则返回false

alert( “1” == “1” );//true

alert( 1 == “1” );//true

只会判断左边和右边的值是不是相同，相同则返回true。不判断数据类型是否相同

alert( 1 === “1” )//false

要判断值是否相同同时判断类型是否相同，只要有一个不同，就返回false

alert( “1” !== 1 );//true
alert( “1” !== “s” );//true
alert( “1” !== “1” );//false
!==
只要值或者类型有一个是不同的,就返回true,如果类型和值都相同,则返回false
逻辑运算符

逻辑运算符
&& 与、|| 或、! 否

&&与（并且）
    如果左边为真，则返回右边，
    如果左边为假，则返回左边。

||或（或者）
    如果左边为真，则返回左边，
    如果左边为假，则返回右边。
!否
    把后面的运算数转成布尔值然后取反

判断a 是否大于1小于5
var a = 3;
if( a>1 && a<5 ){
alert( “大于1小于5” );
}

alert( true && true);//如果左边为真，则返回右边
alert( true && 1 );//如果左边为真，则返回右边
alert( false && 1 );//如果左边为假，则返回左边
alert( 0 && 10 );//如果左边为假，则返回左边

alert( true || 1 );//如果左边为真，则返回左边
alert( 1 || 2 );//如果左边为真，则返回左边
alert( false || 2 );//如果左边为假，则返回右边
alert( false || 0 );//如果左边为假，则返回右边
alert( 0 || 0 );//如果左边为假，则返回右边

alert( 1<2 || 3>4 );//true
alert( 1>2 || 3>4 );//false

alert( !0 );//true
alert( !1 );//false
alert( !!3 );//true

三目运算符

判断条件?语句1:语句2
判断条件成立，执行语句1
判断条件不成立，执行语句2

var a = 1;
    if( a == 1 ){
        alert("是1");
    }else{
        alert( "不是1" );
    }
    a == 1 ? alert("是1"):alert("不是1");


var onoff = true;
    var a;
if( onoff ){
    a = 1;
}else{
    a = 2;
}

alert( onoff ? 1 : 2 );

    a = onoff ? 1 : 2;
alert(a);
